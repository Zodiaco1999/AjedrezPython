Ajedrez por Consola
Este proyecto es una implementación de un juego de ajedrez jugable desde la consola, desarrollado en Python. Permite a dos jugadores enfrentarse siguiendo las reglas oficiales del ajedrez, mostrando el tablero y gestionando los movimientos, turnos y condiciones especiales como jaque, jaque mate, ahogado, coronación, repetición y regla de los cincuenta movimientos.

Características
Interfaz por consola simple y clara.
Soporte para todas las reglas oficiales del ajedrez.
Detección de jaque, jaque mate, ahogado, repetición y cincuenta movimientos.
Gestión de turnos y validación de movimientos.
Mensajes informativos y renderizado del tablero en texto.

Requisitos
Python 3.7 o superior
Ejecución
Clona este repositorio.
Ejecuta el archivo principal:
python main.py
